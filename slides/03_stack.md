## Stack

- Abstrakter Datentyp
- LIFO-Prinzip (Last In, First Out)
- Push: Rücksprungadresse wird auf den Stack gelegt
- Pop: oberster Eintrag wird entfernt (Pop) und die CPU springt zur gespeicherten Adresse zurück
- CPU-Stack (Call Stack)
    - im RAM
    - Verwaltiung mittels Registern, z. B. dem Stack Pointer
